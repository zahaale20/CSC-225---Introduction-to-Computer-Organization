### CSC 225, Assignment 1
A working implementation of functions for binary and hexadecimal numbers.

Binary functions: 
- add
- negate
- subtract
- multiply
- binary_to_decimal
- decimal_to_binary

Hexadecimal functions:
- binary_to_hex
- hex_to_binary

Required files:
  * `binary.py`
  * `hexadecimal.py`

Optional files:
  * _none_
